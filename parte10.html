Configurar IP de los equipos

Una dirección IP (Protocolo de Internet) es una etiqueta numérica única asignada a cada dispositivo conectado a una red de computadoras que utiliza el protocolo IP. Esta dirección se utiliza para identificar y localizar los dispositivos en una red, permitiendo que se comuniquen entre sí y compartan información. Las direcciones IP pueden ser versiones IPv4 (por ejemplo, 192.168.1.1) o IPv6 (por ejemplo, 2001:0db8:85a3:0000:0000:8a2e:0370:7334) y son esenciales para el funcionamiento de Internet y las redes locales.
