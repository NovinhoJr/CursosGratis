Introducción al direccionamiento IPv4

El direccionamiento IPv4 es un sistema que se utiliza para asignar direcciones IP en Internet. IPv4 (Protocolo de Internet versión 4) utiliza direcciones IP compuestas por 32 bits, representadas en el formato decimal con cuatro números separados por puntos (por ejemplo, 192.168.1.1). Cada dispositivo conectado a Internet, como computadoras, servidores y routers, recibe una dirección IPv4 única que se utiliza para identificarlo y enrutar datos a través de la red. Sin embargo, debido a la creciente demanda de direcciones IP, IPv4 se está agotando, lo que ha llevado al desarrollo de IPv6 con direcciones IP de 128 bits para resolver esta limitación.
